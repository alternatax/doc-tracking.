ตั้งค่า Firebase

ทำครั้งเดียว ใช้ได้ตลอด

📋 ขั้นตอน (ทำครั้งเดียว ~10 นาที):

  1. ไปที่ console.firebase.google.com → Add project
  2. เมนูซ้าย → Build → Authentication → Get started → แท็บ Sign-in method → เลือก Google → Enable → Save
  3. เมนูซ้าย → Build → Realtime Database → Create database → Start in test mode → คัดลอก URL
  4. ⚙️ Project Settings → General → Your apps → Add app → Web → Register → คัดลอก apiKey และ projectId
  5. Authentication → Settings → Authorized domains → Add domain → ใส่ domain ที่เปิดแอป

🔒 Security Rules ที่ต้องตั้งใน Realtime Database → Rules:

{
  "rules": {
    ".read": "auth != null",
    ".write": "auth != null"
  }
}

→ Realtime Database → Rules → วางโค้ดด้านบน → Publish
